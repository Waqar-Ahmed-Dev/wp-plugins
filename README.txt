A simple plugin to build and display the user listing for your website.

 == Description ==

 The Inpsyde Users List plugin allows you to list users from API call on your website. You get an easy-to-use interface
which allows you to edit the following fields.

You'll use the `[list-users]` shortcode within a page created name "Inpsyde Users List". 


== Installation ==

 1. Upload the `inpsyde-users` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[list-users]` in a page or post

 == Frequently Asked Questions ==

= My Users List won't show up on my site! =
Alright, here's a few things to try:
